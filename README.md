# Power BI Data Analytics Project

## Overview
This repository contains a Power BI data analytics project focused on building interactive reports and dashboards, supported by project documentation and deliverables.

## Tools and Technologies
- **Microsoft Power BI Desktop**: Data modeling, report building, and dashboard design.
- **DAX (Data Analysis Expressions)**: Calculated measures and KPIs.
- **Power Query (M)**: Data cleaning, transformation, and shaping.
- **Data Visualization Best Practices**: KPI cards, trend analysis, and interactive slicers.
- **Project Documentation**: Supporting report and assignment documentation (PDF/TXT).

## Achievements
- Built an end-to-end analytics solution combining data transformation, modeling, and reporting.
- Delivered an interactive dashboard with KPI tracking and trend analysis for decision support.
- Documented project outcomes and methodology for reproducibility.

## Results
- A ready-to-open **Power BI report** (`Power BI Project.pbix`) showcasing the final analytics solution.
- A written **project report** (`Project Report.pdf`) describing objectives, approach, and findings.
- An **assignment brief** (`Assignment.txt`) detailing requirements and scope.

## Getting Started
1. Open `Power BI Project.pbix` in Power BI Desktop.
2. Review `Project Report.pdf` for methodology and results.
3. Reference `Assignment.txt` for original project requirements.
