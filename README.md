# Power BI Data Analytics Project

## Overview
This repository contains a Power BI report and supporting documentation for analyzing a B2B sales pipeline dataset. The analysis explores team performance, individual sales agent performance, quarter-over-quarter trends, and product win rates.

## Key Questions Addressed
- How is each sales team performing compared to the rest?
- Are any sales agents lagging behind?
- Are there any quarter-over-quarter trends?
- Do any products have better win rates?

## Contents
- `Power BI Project.pbix`: Power BI report/dashboard file.
- `Project Report.pdf`: Written report summarizing findings.
- `Assignment.txt`: Source problem statement and analysis prompts.

## How to Use
1. Open `Power BI Project.pbix` in Power BI Desktop.
2. Review report pages and visuals for team, agent, trend, and product performance.
3. Reference `Project Report.pdf` for narrative findings and conclusions.

## Requirements
- Power BI Desktop (latest recommended).

## Notes
If you plan to extend the analysis, consider importing additional quarterly data or adding new dimensions such as region or customer segment.
